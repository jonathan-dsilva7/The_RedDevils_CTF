## Setup Instructions

### Requirements
- VirtualBox (recommended)

### Steps
1. Download the OVA file from the link below
2. Open VirtualBox → **File → Import Appliance**
3. Select the downloaded OVA file and click **Next**
4. Leave all settings as default and click **Import**
5. Start the machine
